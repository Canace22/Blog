Git常用命令

Jul 8, 2018 | 技术 | 阅读

  1. 添加配置信息: git config <配置信息名>

  2. 删除配置信息: git config unset <配置信息名>

  3. 查看配置信息: git config -l

  4. 提交代码: git status查看状态 => git add <文件名> => git commit -m <描述> => git pull origin <远程主机名> :<远程分支名> => git push <分支>
