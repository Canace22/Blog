一、快速上手

1. 安装

npm i -g typescript

2. 编译上下文

(1) tsc --init 创建 tsconfig.json 文件, 创建好之后的内容长这样的

(2) 显式指定需要编译的文件

(3) 使用 include 和 exclude 选项来指定需要包含的文件和排除的文件

(4) 继承来自其他文件的配置项

3. 开始编译

执行 tsc 会编译项目中指定的 ts 文件(没有配置则是全部)到指定目录

二、类型简介

1. 原始类型

js 中的 string、number、boolean 等原始类型可以直接作为类型注释

2. 特殊类型

(1) any: 可以被赋值给任意类型，也可以被任意类型赋值

(2) null、 undefined: 可以被赋值给任意类型

(3) void: 函数返回值为空

3. 元素类型相同的数组

数组的类型注释有两种方式

(1) 原始类型 + []

(2) 数组泛型 Array

4. 元素长度已知的，每个元素类型已知的数组

这类数组可以用 ts 元组来注释

5. 接口

接口相当于给一个类型定义集合，用到该接口注释类型的，必须遵从里面的规则, 更多细节可以查看文档

6. 泛型

函数参数未定，需要函数返回值与传入参数类型一致时，会用到泛型, 更多细节请查看文档

7. 联合类型

当类型可能有多个时，可以选择用联合类型
